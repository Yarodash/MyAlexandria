# MyAlexandria

Telegram bot that aggregates blog posts from MyAlexandriya.blogspot.com and delivers them to subscribers with interactive subscription management and automatic notifications.

![Cat](https://images.unsplash.com/photo-1574158622682-e40e69881006?w=400&h=300&fit=crop)

**Tech Stack:** `Python` `Telegram Bot API` `BeautifulSoup4` `Web Scraping`
